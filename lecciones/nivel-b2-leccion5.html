Phrasal Verbs Avanzados

Más combinaciones esenciales para sonar natural.

1. Phrasal verbs comunes

Put offPosponer
Carry onContinuar
Bring upMencionar / Criar
Break downDescomponerse / Colapsar
Figure outDescifrar / Entender
Run intoEncontrarse con alguien
Set upConfigurar / Establecer

Ejemplos:

  • Let's not put off this decision. (No pospongamos esta decisión.)
  • I ran into an old friend yesterday. (Me encontré con un viejo amigo ayer.)
  • I need to figure out this problem. (Necesito descifrar este problema.)
