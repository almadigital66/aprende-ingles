Tercer Condicional (Third Conditional)

Para hablar de situaciones hipotéticas del pasado que ya no se pueden cambiar.

1. ¿Cuándo se usa?

Se usa para imaginar un resultado diferente de algo que ya pasó — arrepentimientos o especulación sobre el pasado.

💡 Estructura: If + pasado perfecto (had + participio), would have + participio. Ejemplo: If I had studied, I would have passed.
If I had studied, I would have passedSi hubiera estudiado, habría aprobado
If she had known, she would have calledSi ella hubiera sabido, habría llamado

Ejemplos:

  • If I had known, I would have come. (Si hubiera sabido, habría venido.)
  • If we had left earlier, we wouldn't have missed the flight. (Si hubiéramos salido antes, no habríamos perdido el vuelo.)
💡 Dato clave: este condicional habla de algo que ya no se puede cambiar — es puramente hipotético sobre el pasado.
