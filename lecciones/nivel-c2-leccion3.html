Condicionales Mixtos (Mixed Conditionals)

Cuando una condición pasada afecta el presente, o viceversa.

1. Pasado → Presente

💡 Estructura: If + pasado perfecto, would + verbo base. Ejemplo: If I had studied medicine, I would be a doctor now.
If I had studied medicine, I would be a doctor nowSi hubiera estudiado medicina, sería doctor ahora

2. Presente → Pasado

💡 Estructura: If + pasado simple, would have + participio. Ejemplo: If I weren't so busy, I would have gone.
If I weren't so busy, I would have goneSi no estuviera tan ocupado, habría ido
  • If she hadn't moved, she would still be my neighbor. (Si ella no se hubiera mudado, todavía sería mi vecina.)
