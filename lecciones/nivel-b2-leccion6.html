Used to / Would: Hábitos del Pasado

Para hablar de cosas que hacías antes, pero ya no.

1. "Used to" — hábitos y estados pasados

I used to smokeYo solía fumar (ya no)
She used to live in PeruElla solía vivir en Perú
We used to be friendsSolíamos ser amigos
💡 Dato clave: "Used to" sirve tanto para acciones repetidas como para estados (ser, vivir, tener). "Would" solo funciona para acciones repetidas, no para estados.

2. "Would" — acciones repetidas en el pasado

Every summer, we would go to the beachCada verano, íbamos a la playa
My grandmother would tell storiesMi abuela solía contar historias
  • I used to be shy. (Correcto: estado, con "used to")
  • I would be shy. (Incorrecto para este uso — "would" no funciona con estados)
